= Classy

Classy is an academic platform for better student scheduling, planning, and learning. 

We hope our tool can encourage exploration and ownership over ones education.

== Installation
Classy is built using Ruby on Rails. 

To install the application's Ruby packages:

  bundle install
  
Then to prepare the database:
  
  rake db:migrate
  
Finally run Classy on port 3000, run: 
  
  rails s
